本目录用于保存各页面独立设计稿和页面级交互约束。总体视觉规范见 `../FRONTEND_DESIGN_STYLE.md`。

## 当前设计稿

- `public-home.md`：公开首页设计稿，覆盖 AI-first 品牌入口、双角色 CTA、AI 质检预览和响应式约束。
- `public-home-style-summary.md`：公开首页视觉风格总结，归纳当前平台色彩、字号、组件和动效基线。
- `public-home-design.md`：公开首页设计文档，覆盖首页信息架构、首屏方案、模块结构、CTA、响应式和 GSAP 动效策略。
- `public-home-prototype.md`：公开首页原型说明，覆盖文字版结构、布局、动效与组件拆分。
- `public-solutions.md`：公开套餐方案页设计稿，覆盖五档员员套餐、怎么选判断列表和公开 `/publish` 入口兼容策略。
- `public-help.md`：公开帮助手册设计稿，覆盖参考 Gitea Docs 的文档站布局、移动端角色筛选、全文搜索、章节目录、正文文章和 FAQ 约束。
- `owner-production.md`：Owner 生产页总体设计稿，覆盖数据集、模板和发布任务的共性约束。
- `owner-dataset-management.md`：数据集管理列表页与数据集修改子页面设计稿。
- `owner-task-management.md`：任务管理列表页与新建/修改任务子页面设计稿。
- `owner-task-publish-ai-assistant.md`：任务发布向导内 AI 浮窗助手、完整草稿上下文、结构化发布变更确认和本地应用设计稿。
- `owner-template-designer.md`：模板管理列表页、Designer 子页面与 Renderer 预览子页面设计稿。
- `owner-template-ai-assistant.md`：模板 Designer 内 AI 浮窗助手、Provider 复用、结构化变更确认和本地应用设计稿。
- `organization-management.md`：企业管理页面组总体设计稿，统一企业信息、资源配置、公告通知、操作日志的导航、权限、联动、降级和 Ant Design 实现边界。
- `organization-announcements.md`：公告通知、角色/成员/任务分发、接收人预览、审核提醒和导出提醒页面设计稿。
- `organization-audit-logs.md`：操作日志、审计筛选、日志详情 Drawer、风险等级和事件覆盖矩阵页面设计稿。
- `organization-profile.md`：企业信息、企业认证、企业资料维护、成员统计和最近变更页面设计稿。
- `organization-resource-config.md`：员员与额度、积分管理、AI 资源、AI provider、生产开关和资源治理页面设计稿。
- `platform-workbench.md`：平台运营工作台，覆盖经营总览、结算流水、提现处理、认证审核和平台规则。
- `team-member-management.md`：人员管理主表格、添加成员、成员详情与成员操作设计稿。
- `reviewer-manual-review.md`：Reviewer 人工审核三栏工作台、AI 预审队列、直接修订、批量处理、审计时间线和侧栏入口设计稿。
